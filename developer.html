Developer API

Integrate professional voice generation into your applications with our simple REST API.

POST Generate Audio

Convert text to speech using any of our premium voices.

POST /api/tts
Content-Type: application/json

{
  "text": "Hello world",
  "voice": "brian",
  "format": "wav"
}

GET List Voices

Get all available voices with metadata.

GET /api/voices

Response:
{
  "voices": [
    { "id": "brian", "name": "Brian" },
    { "id": "andy", "name": "Andy" }
  ]
}

POST Voice Clone

Create custom voice from audio sample.

POST /api/clone
Content-Type: multipart/form-data

// Upload audio file + text
file: <audio.wav>
text: "Your script here"

GET Usage Stats

Check your character usage and plan limits.

GET /api/usage
Authorization: Bearer <token>

Response:
{
  "characters_used": 12500,
  "characters_limit": 50000
}
